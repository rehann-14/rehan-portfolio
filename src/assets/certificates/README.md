# Certificates Folder

Place your certificate image files here.

**Supported formats:** JPG, PNG, WebP

**How to add a certificate:**
1. Place your certificate image in this folder (e.g., `google-dm.jpg`)
2. Open `src/data/certificates.js`
3. Update the `image` field for the relevant certificate:
   ```js
   image: '/src/assets/certificates/google-dm.jpg'
   ```

**Naming convention:**
- `google-analytics.jpg`
- `hubspot-content.jpg`
- `meta-social.jpg`
- etc.
