# Figma Project Images

Place your Figma project thumbnail images here.

**How to add thumbnails:**
1. Export your Figma frame as JPG/PNG
2. Place the image here (e.g., `ecommerce-app.jpg`)
3. Open `src/data/projects.js`
4. Update the `thumbnail` field:
   ```js
   thumbnail: '/src/assets/images/figma/ecommerce-app.jpg'
   ```

**Recommended size:** 800x600px or 1200x900px
